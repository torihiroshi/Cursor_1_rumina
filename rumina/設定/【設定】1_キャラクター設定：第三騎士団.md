# **キャラクター設定：第三騎士団**

## **リエル (Riel)**

* **役割** : 主人公。エリンディルの最後の生き残り。ロディニア魔法使い団に所属し、現在は第三騎士団へ派遣されている魔女。  
* **年齢** : 17歳。  
* **外見** : 常闇の結晶果の影響で人間離れした美貌を持つ。透き通るような白い肌、血のように鮮やかな赤い唇、長く濃密なまつ毛に縁どられた、宝石のように輝く深い紅色の瞳（感情の高ぶりや闇魔法使用時に強く発光）。髪は肩までの長さで、元の黒髪から太陽光を溶かし込んだような淡い赤茶色へと変化している。服装はマグヌスから与えられた黒と青を基調とした魔法戦闘服（防御・保温・魔力増幅・自己修復機能付き）の上に、状況に応じて灰色のマントや毛皮のコートを羽織る。足元は頑丈な防寒ブーツ。母が作ってくれた赤いローブは戦闘で損傷が激しいが、自身で拙いながらも修繕し、大切に保管している。【追記】エレナから譲り受けた魔除けの銀のネックレス（青い魔石付き）を常に身に着けている。  
* **性格** : イリアス国上層部と、里を襲撃した兵士たちへの燃えるような復讐心が行動原理の核。目的のためには非情な手段（襲撃、殺害、デスティール）も厭わない冷徹さを身につけつつあるが、それは後天的なものであり、根は感受性が豊か。滅びの遺跡に美しさを見出すような詩的な感性や、他者の苦しみへの共感性も残っている。アバカンでの生活を通じて、エレナやカイルといった仲間との間に芽生え始めた絆や温かさに戸惑いながらも、それを大切にしたいと感じ始めている。ロディニアの進んだ生活魔法への純粋な興味や、新しい服への関心など、17歳の少女らしい一面も覗かせる。しかし、常闇の結晶果の影響による昏い衝動、特にデスティール（生命力吸収）への抗いがたい誘惑と、過去に犯した罪（5章での父娘襲撃、特に少女への行為とその生死）への罪悪感に常に苛まれている。【追記】 戦争が近づくにつれ、デスティールによる力獲得の機会が増えることを、恐怖と共にどこかで期待してしまっている自分に気づき、自己嫌悪に陥る。【追記】 エリンディルでの襲撃のトラウマから、激しい雷鳴や炸裂音に対してPTSDのような反応を示すことがある。【追記】 ルミナ信仰については、里の壊滅と両親の死を目の当たりにし、「ルミナ様は私たちを見捨てた」と感じており、信仰心を失っている。【追記】 魔法以外の実用的なスキル（裁縫など）は苦手。【追記】9章でエレナからネックレスを譲り受けたことで、デーモンの直接的な声からは一時的に解放されているが、心の奥底の不安や葛藤は続いている。  
* **背景** :  
  * イリアスの辺境、深い森の奥に隠された魔法使いの里「エリンディル」で生まれ育つ。【追記】 子供の頃はレヴィタスを使い、よく一人で里を抜け出して森を探検し、食べられる虫や野草の知識を得るなど、活発な面もあった。  
  * 17歳の誕生日を間近に控えたある日、イリアス正規軍の襲撃により里は壊滅。両親（エララ、トルヴィン）も目の前で惨殺される。自身は地下通路から辛くも脱出。  
  * 母（エララ）: 熟練した織物の職人で、里の女性たちのまとめ役。リエルの深紅の魔法使いのローブ（内側には保温のための毛皮付き）は、彼女が娘の成長と安全を願って特別に織り上げ、複雑な守護の魔法を付与したもの。穏やかだが芯が強く、外の世界の情報を得て里の将来を憂慮していた節もある。  
  * 父（トルヴィン）: 主に畑仕事に従事し、里の食料自給を支える。口数は少ないが思慮深く、素朴で温厚な人柄。畑仕事の合間に木陰で古い書物（特に聖典）を読むのを好み、リエルにはルミナ神話の物語だけでなく、そこから読み取れる教訓（世界の調和、光と闇のバランス、慈悲の心など）を、自身の言葉で優しく語り聞かせていた。  
  * 里を脱出した後、山頂の遺跡でデーモンに遭遇し、常闇の結晶果を強制的に食べさせられ、内に秘められていた「魔界の血」が覚醒する。  
  * その後、デーモンハンターのパーブルと出会い、束の間の同行の後に別れる。ロディニアへ入国後、偶然にも第三騎士団と共闘したことがきっかけで、マグヌスの目に留まり、ロディニア軍（ロディニア魔法使い団所属・第三騎士団派遣）に身を置くことになる。現在はアバカンで、マグヌスや騎士団の仲間たちの下で訓練に励む日々を送る。  
  * 里が「世界のバランサー」として、結晶果の管理という重要な役割を担っていたこと、そしてその血筋が持つ真の意味についてはまだ知らない。  
* **能力** :  
  * **闇魔法（古文書変容前）** : 常闇の結晶果と「魔界の血」により発現。強大な威力を持つが、魔力消費が激しく、精神への負荷も大きい。  
    * オーブネス: 基本的な闇のエネルギー球。訓練により威力・精度が向上。回転オーブネス（マグヌス伝授）は軌道を変化させ、防御を掻い潜る応用技。  
    * シャドウ・バインド: 影を実体化させ操る。物理的な拘束、攻撃（影の手、影の剣など）、防御（衝撃吸収、壁）、物品運搬、自身の補助（移動、マッサージ）に応用可能。リエルは眷属討伐任務（8章）で防御（衝撃吸収）、拘束（枝）、救助（フィン）に活用。魔力消費が大きい。【追記】 習熟度が低いため、力は強いが動きがぎこちなく、力加減が非常に難しい（物を壊したり、意図しない動きをしたりする）。リエルにとっては「力の強いあほの子」のような、扱いにくいペットに近い感覚。【更新】5章で入手した三本爪の杖を使うと、影の手が三本爪の形になり、力が増幅されるだけでなく、リエルの意思との繋がりが強化され、以前よりも精密な操作が可能になる。ただし、杖の使用は魔力消費をさらに増大させるため、切り札的な扱いとなっている。  
    * デスティール: 対象の生命エネルギー（魂？）を吸収し、自身の力や治癒力に変える。極めて強力かつ危険な魔法。使用時には強い全能感や快感を伴うが、精神汚染のリスクが高く、無差別な使用衝動に駆られる。古文書の解読を進める鍵でもあるらしい。【追記】 吸収対象によって得られる感覚が異なり、人間の魂（特に絶望や恐怖に満ちたもの）からはより強い快感と力を得るが、動物や眷属からはそれほどではない（ただし、力の回復効果はある）。  
    * 強化イグニス: 紫がかった魔界の炎。通常のイグニスより高火力で、浄化されにくい特性を持つ。  
  * **【追記】闇魔法（古文書変容後 \- 11章以降）**: 原初の闇の声と古文書の変容により、デーモンの影響が強かった旧来の闇魔法（オーブネス、シャドウ・バインド、デスティール）は**使用不可能**となる。強化イグニスも通常の炎の性質に戻る。代わりに、古文書から世界のバランスを保つための新たな力が目覚め始める。  
    * **【New】リエルの「調和のための力」について**  
      * **力の性質と目的**: リエルが目覚める「調和のための力」は、従来の闇魔法が直接的な破壊や殺傷を目的としていたのとは対照的に、「戦いを収める」ことを主眼とする。具体的には、邪悪な力に操られている人物を殺害するのではなく、その支配力のみを無力化するなど、不必要な犠牲を避けることを目指す力である。  
      * **ルミナ教の魔法との関連性**: この「調和のための力」は、リエルの故郷エリンディルで信仰されていたルミナ教の魔法の根源的な性質と深く関わる。ルミナ教の魔法は、本来、光と闇という相反する要素を内包し、それらが調和することで真価を発揮するものであった。リエルの新たな力は、この本来のルミナ教の魔法の姿を体現するものと言えるかもしれない。  
    * **ステラ・オーブ (Stella Orb) \- 光・闇**: 眷属を浄化し、闇に染まりきっていない存在を元の姿に戻す可能性を秘めた、光と闇の属性を併せ持つエネルギー球。  
    * **スピリトゥス・アストラム (Spiritus Astrum) \- 光・闇**: 中心が白く輝き、周囲を紫色のオーラが包む、ウサギのような精霊（あるいは使い魔）を召喚する。偵察、伝令、限定的な治癒、さらには巨大化して剣を咥えて戦うなど、戦闘補助も可能。シャドウ・バインドに代わる力。  
    * **トワイライト・パージ (Twilight Purge) \- 光・闇**: 救済不能なほど深く闇に堕ちた魂を強制的に吸収し、浄化・解放して原初の闇へと還す魔法。デスティールとは異なり、使用者への精神汚染リスクは低い（あるいは性質が異なる）。この魔法の使用によって、古文書の真の解読が進む。  
  * **浮遊魔法（フロート・ウィング）** : 結晶果の影響で基礎的なレヴィタスが大幅に強化されたもの。長時間の飛行や、オーブネス噴射による急加速・方向転換など、戦闘機動にも応用可能。ただし、精神状態や疲労度によって安定性に欠け、常に自在に使えるわけではない。  
  * **感覚強化** : 闇視能力、動体視力、遠方視力などが大幅に向上。魔力感知能力も鋭敏になり、微細な魔力の流れや種類、感情の機微（殺気、敵意など）を読み取れる。  
  * **自然四属性魔法（訓練中）** : マグヌスの指導で基礎を学習中（イグニス、トール、エーギル、ヨルズ）。魔力制御の向上を目的とする。  
  * **防御魔法（訓練中）** : テトラ・シールドの基礎を学習中。潜在的に高い魔法防御力を持つ可能性が示唆されている（眷属の攻撃を受けても致命傷に至らないなど）。  
  * 【追記】**古文書感応** : 焼け焦げたデーモンの古文書と精神的にリンクしており、デスティール使用など特定の条件下で新たな知識（魔法、文字解読）が流れ込んでくる。強力な闇魔法（例：ワード・オブ・ダークミスト）に遭遇した際に、その名称や危険性を本能的に理解することがある。  
  * 【New】**三本爪の杖** :  
    * 入手経緯 : 5章にて、ロディニアの丘陵地帯にある古代遺跡の地下で発見。デーモン曰く「かつて同胞が使っていたもの」。  
    * 外見・特性 : 柄は黒檀のように見えるが驚くほど軽く、先端は黒い石のような質感で三本の鋭い爪が開いた形状。長さは約1m。鉄のように硬い。  
    * 効果 : 杖を通して魔法を使うことで、全体的な魔法の威力が向上する。特に闇魔法との親和性が高く、制御が難しかったシャドウ・バインドも、杖を使うことでより精密な操作が可能になる。（古文書変容後は、新たな力との親和性が示唆される）  
    * 制約 : 魔力消費が大幅に増大するため、常時使用は難しく、リエルにとっては切り札的な扱いとなっている。 【追記】9章の会議で、デーモンからの干渉を防ぐため、エレナの家宝である魔石を装着することが決定した。  
* **今後の可能性** : イリアスとの戦争という過酷な現実の中で、復讐心と仲間との絆、そして自身の内に潜む闇との間で激しく揺れ動きながら成長していく。ヴィクトルやデーモンの真の目的、そして自身の血と力の意味を知った時、彼女がどのような選択をするのか。最終的には、ヴィクトルが示す「混沌と進化」でも、エレナが体現する「秩序と救済」でもない、リエル自身の答えを見つけ出し、光と闇の狭間で生きる道を選ぶかもしれない。

## **エレナ・ラムスドルフ (Elena Ramusdorf)**

* **役割** : ロディニア第三騎士団・副官。貴族。リエルの保護者的存在であり、同時に監視者でもある。経験豊富なデーモンハンター。  
* **年齢** : 20代後半～30代前半と思われる。  
* **外見** : 輝く金髪と澄んだ碧眼を持つ、理知的な美女。常に穏やかな微笑みを絶やさないが、その奥には冷静さと、時折デーモンハンターとしての鋭い警戒心が覗く。長身で、鍛え上げられたしなやかな体躯を持つ。  
* **服装** : 第三騎士団の副官用制服（白と青基調）。腰には愛剣であるレイピア「スノウタリア」（ 9章で魔石装着により強化済 ）を佩いている。首には父から譲り受けた魔除けの銀のネックレスを身に着けていたが、 9章でリエルに譲り渡した。  
* **性格** : 基本的に穏やかで礼儀正しく、常に「ですます」調で話す。面倒見が良く、部下や弱者に対する思いやりに溢れ、カイルや守備兵など多くの者から深く信頼されている。しかし、戦闘においては冷静沈着かつ非情なまでの合理性を発揮する戦術家。精密な氷魔法による無力化や、急所への的確な一撃を好む。リエルに対しては、保護すべき少女としての慈しみと、その境遇への同情を感じているが、同時に彼女が持つ強大な闇の力と、それに引き寄せられるデーモンの気配に対し、デーモンハンターとしての血が強い警戒心を抱かせている。【追記】 大浴場でのリエルの瞳の発光（ 7章 ）や、セルトナの森でのデスティール抑制時のスノウタリアの反応（ 8章 ）などから、リエルへの懸念を深めている。父パーブルの長年の失踪が深いトラウマとなっており、家族や仲間を失うことへの恐れが強い。【追記】 カイルの無茶な行動には呆れつつも、冷静にフォローする保護者的な一面もある。 【追記】9章でリエルをデーモンの影響から守ることを決意し、ネックレスを譲渡。また、スノウタリアを通じてデーモンの声を聞き、剣が狙われていることを知る。  
* **背景** : かつては名門だったデーモンハンター貴族「ラムスドルフ家」の現当主。しかし、父パーブルが実体を持つデーモンを追って外国へ放浪し、長年音信不通となったことで家は没落。家の再興と、父の安否確認（そして、父が果たせなかったデーモン討伐の完遂）が彼女の密かな悲願でもある。騎士団での高い地位は、彼女自身の能力と努力によるもの。 【追記】子供の頃は家庭教師をつけられるほどの経済的余裕はなく、騎士団に入ってから独学で様々な知識を学んでいる（9章）。  
* **能力** :  
  * **精密氷魔法** : エーギル系の魔法を得意とし、特に微細な氷片や氷柱を高速かつ正確に操る技術に長ける。カウンター攻撃や、敵の動きを阻害・無力化する戦術を得意とする。  
  * **光魔法** : デーモンハンターの家系として、対魔・浄化の光魔法も習得している。ソル（基本光）、パラゾニウム・ルクス（光の矢/小剣）、クリスタル・ロータス（浄化結界）、そして切り札であるワード・オブ・ホワイトミスト（広範囲絶対零度攻撃）など。  
  * **剣術** : レイピア「スノウタリア」を用いた剣技。スノウタリア自体が魔力、特に闇の力に反応し、常闇の眷属に対して特攻効果を持つ。 9章で魔石装着による強化が行われ、一日一発限定で強力な光の刃を放つ聖光刃（せいこうじん）が使用可能になった。また、魔石によりデーモンからの干渉を防ぐ効果も付与された。  
  * **指揮・戦術** : 副官としてガルド団長を補佐し、部隊の指揮や作戦立案にも優れた能力を発揮する。特に冷静な状況分析と、被害を最小限に抑えるための精密な戦術指示に長ける。  
* **今後の可能性** : イリアスとの戦争、そして激化するデーモンや眷属との戦いを通じて、デーモンハンターとしての宿命と向き合う。リエルとの関係も、保護者と被保護者から、共に戦う戦友へと変化していく中で、彼女自身のトラウマや価値観も変化していく可能性がある。最終決戦において父パーブルと再会。戦後は、騎士団に留まるか、あるいは故郷に戻り、ラムスドルフ家再興と、衰退したルミナ教の再興（司祭として？）に尽力し、信仰を通じて世界の安定を取り戻す道を歩むかもしれない。

## **【追記】ヴァレリウス・インヴィクトゥス (Valerius Invictus)**

* **役割** : エレナの先輩にあたるデーモンハンター。第三騎士団所属の聖騎兵隊副隊長（あるいは実質的な隊長代理）。  
* **年齢** : 不明（エレナより年上と思われる）。  
* **外見** : 不明。  
* **服装** : 不明（聖騎兵隊の制服？）。  
* **性格** : プライドが高く、実力主義者（実力が高いものが上に立つべきと考えている）。貴族ではないが、貴族以上に貴族らしい振る舞いをすることもある。野心家であり、出世のためなら危険な最前線の任務も厭わない。エレナに対しては、その家柄や立場、あるいは能力に対してライバル心を抱いている。  
* **背景** : 父はかつてパーブル（エレナの父）と共に戦ったデーモンハンター。しかし、パーブルのみが貴族として叙勲されたため、ヴァレリウス自身は貴族ではない（そのことにコンプレックスや反発心を抱いている可能性あり）。父譲りのデーモンハンターとしての高い実力を持つ。 【追記】苗字の「インヴィクトゥス」は、正式に叙勲されなかった彼が、自身の不屈の意志と実力を示すために自ら名乗っているものである。  
* **能力** :  
  * **戦闘能力** : 最前線での戦闘経験が豊富で、その実力はエレナ以上ともされる。デーモンハンターとしての対魔戦闘技術に長ける。 【追記】武器戦闘に長け、防御魔法（特にパラゾニウム・シールドのような光の盾）を得意とする。  
  * **魔法** : 不明（光魔法や対魔系の魔法を得意とする可能性が高い）。  
  * **指揮能力** : 聖騎兵隊を率いる（エレナ不在時など）。  
* **現状** : 主にエレナとは別の戦線（国境付近など）で、聖騎兵隊を率いて常闇の眷属関連の任務についていることが多い。  
* **今後の可能性** : 戦争が激化する中で登場し、エレナやリエルと共闘、あるいは対立する可能性がある。彼の野心やエレナへのライバル心が、物語に新たな緊張感をもたらすかもしれない。

## **カイル・ベラルド (Kyle Berard)**

* **役割** : ロディニア第三騎士団所属の騎士。リエルの最初の友人。自称「特攻隊長」。ムードメーカー。  
* **年齢** : 17歳。リエルとほぼ同い年。  
* **外見** : 健康的な茶色の髪と茶色の瞳を持つ、快活な印象の青年。人懐っこい笑顔が特徴。鼻の横には、訓練か実戦で負ったと思われる小さな傷跡がある。  
* **服装** : 第三騎士団の標準的な制服を着用しているが、動きやすさのためか、よく袖をまくり上げている。腰には愛用のバスタードソード。非番の時やサロンなどでは、毛皮の襟がついた革ジャケットを好んで着ている。【追記】 5章 の眷属戦でジャケットの背中を燃やされた。  
* **性格** : 明るく気さくで裏表がなく、誰に対しても物怖じしない純粋な性格。仲間思いで、特にリエルに対しては、彼女の境遇を気遣い、積極的に話しかけて元気づけようとする。魔法の才能は低い（本人も自覚）が、剣術に関しては天賦の才と絶対的な自信を持つ。戦闘においては、理論よりも本能的な感覚（見切り）で動き、危険な状況でも臆さず前に出る勇気を持つ。【追記】 武器を失っても強がって最前線に出たり、回避に夢中になって油断したりするなど、やや無鉄砲で調子に乗りやすい一面もある（内心では焦りや恐怖を感じていることもあるが、それを表に出さない）。時折、騎士道や友情について熱く語る「男のロマン」的な一面も。 【追記】座学は苦手だが、戦術的な勘は鋭い（9章エレナ談）。  
* **背景** : ロディニアの名門貴族であるベラルド家の三男。長男が家督と領地を継ぐため、次男は別の貴族へ養子に出され、三男であるカイルは騎士団に入隊した。剣術の腕を見込まれての入隊だが、本人は貴族としての責務や将来の領地統治について、内心では多少の悩みやプレッシャーも感じている。家が決めた許嫁がおり、20歳での結婚が予定されている。騎士団内では、第一騎士団に所属する父から譲り受けた部下達を率いており、彼らは歴戦の戦士が多い。カイルは彼らから様々な武器の技術を学んでいる。  
* **能力** :  
  * **卓越した剣術** : バスタードソードを自在に操る。力任せではなく、相手の動きを見切って的確な一撃を加える技巧派。  
  * **見切り** : 戦闘における天性の勘。敵の攻撃を最小限の動きで回避し、瞬時に弱点を見抜く能力。【追記】 武器がない状態でも、魔法攻撃を感覚だけで回避し続けるほどの高い回避能力を持つ。  
  * **投的** : ナイフなどの投擲技術も習得しており、さらに向上させたいと考えている。  
  * **回復魔法（ドライヴ・ヒール）** : 意外にも回復魔法の才能があり、第三騎士団内では「名医」と称されるほど得意。戦闘後の負傷者治療を一手に引き受けることが多い。  
  * **魔法付与剣** : 自身の魔力が低いため、マグヌスや他の魔導士にイグニスなどの属性魔法を剣に付与してもらい、攻撃力を補っている（ただし 5章 の眷属戦で破壊された）。  
  * **武器適性** : バスタードソード以外にも、様々な武器を潜在的に扱える可能性がある。【追記】 予備の剣も携帯している。  
* **今後の可能性** : イリアスとの本格的な戦争を通じて、騎士として、また一人の人間として大きく成長する。リエルとの友情は、彼女の精神的な支えとなり、またカイル自身にとっても、貴族や騎士としての生き方を見つめ直すきっかけとなるかもしれない。許嫁との関係や、ベラルド家での立場も、物語の中で描かれる可能性がある。

## **セリーヌ・ヴァルトナー (Celine Waldner)**

* **役割** : ロディニア第三騎士団所属の弓兵隊隊長。超人的な弓術と破壊的な魔法の使い手。  
* **年齢** : 30代前半と思われる。  
* **外見** : 短く刈り込んだ黒髪と、鋭い緑色の瞳を持つ。精悍な顔立ちには、戦場でついたであろう複数の傷跡が刻まれている。常に自信に満ち溢れた表情をしている。弓兵とは思えないほど筋肉質で、引き締まった体躯を持つ。  
* **服装** : 黒と緑を基調とした、動きやすさを重視した軽装鎧。背中には身長ほどもある巨大な剛弓を背負い、腰には特製の矢が満載された矢筒を下げている。彼女が率いる弓兵隊の多くは、なぜか寒さにもかかわらず半裸に毛皮のベストという、筋骨隆々な男たちで構成されている。  
* **性格** : 非常に豪快で男勝り、竹を割ったような性格。戦いそのもの、特に強大な敵との死闘や、自らの力で敵を蹂躙することに純粋な喜びを見出す戦闘狂。細かい戦略や被害をあまり気にせず、派手な力技を好む傾向がある。仲間思いではあるが、口が悪く、特にカイルとはしょっちゅう軽口を叩き合っている。【追記】 戦闘中でも、仲間の（少し間の抜けた）失敗を見て大笑いするなど、不謹慎とも取れる豪胆さを持つ。ガルド団長には一定の敬意を払っているが、その指示に完全に忠実とは限らない。  
* **背景** : 元々は貴族（ ヴァルトナー 家）の出身だが、領地の管理や貴族社会のしがらみを嫌い、親戚であるガルド家に領地経営を（半ば押し付ける形で）譲渡し、自身は純粋な戦士・弓兵として生きる道を選んだ。意外にも夫と娘がおり、時折、娘の話をすることがある。過去に第一騎士団へ派遣されていたが、その破壊的な戦闘スタイルが問題視されたか、あるいは本人の希望か、第三騎士団へ復帰した。 【追記】9章では、第一騎士団と共に国境付近での魔剣部隊との戦闘任務から帰還したばかり。  
* **能力** :  
  * **超人的な弓術** : 巨大な剛弓を軽々と引きこなし、驚異的な精度と威力で矢を放つ。遠距離からの狙撃だけでなく、中・近距離での速射や曲射も得意とする。  
  * **格闘術** : 弓が使えない状況でも、ヨルズ（重さ）やイグニス（熱）の魔法を付与した拳や蹴りによる強力な格闘戦を行うことができる。  
  * **大魔法** : 広範囲を殲滅する破壊的な魔法も習得している。ドゥール・ネセルト（火＋大地、低威力版を眷属討伐任務で使用）、テラ・フラクチャー（大地、未使用）など。使用には詠唱と魔力充填が必要。  
* **今後の可能性** : イリアスとの戦争において、その圧倒的な火力と破壊力で活躍する場面が多くなるだろう。しかし、その戦闘狂的な側面が、時に味方を危険に晒したり、倫理的な問題を引き起こしたりする可能性もある。

## **フィン (Finn)**

* **役割** : ロディニア第三騎士団所属の斥候。隠密行動、偵察、暗殺、特殊任務のエキスパート。  
* **年齢** : 20代と思われるが、年齢不詳な雰囲気を持つ。  
* **外見** : 色素の薄い茶色の髪と、感情の読みにくい青い目。青白い顔をしていることが多い。背は低く細身で、常にフードを深く被っているため、顔全体が見えることは少ない。存在感が希薄で、気づくとすぐそばにいたり、逆にいつの間にかいなくなっていたりする。  
* **服装** : 黒と灰色を基調とした軽装のローブ。ローブには光学迷彩や気配遮断のための魔法陣（ミラージュ・ヴェール）が織り込まれている。腰には斥候道具（鍵開け、罠解除具、小型望遠鏡など）が入ったポーチ、手には鉤爪（クライミングや近接戦闘用）、足元は音を立てずに歩くための特殊な革靴を履いている。  
* **性格** : 極端に声が小さく、口数も少ない。人前に出ることを好まず、常に部屋の隅や壁際など、「端っこ」にいることを好む。しかし、斥候としての能力はロディニア軍全体でもトップクラスであり、任務遂行中は極めて冷静沈着かつ大胆に行動する。デーモンやその眷属に対しては、個人的な理由からか、静かながらも強い対抗心を抱いている。過去に様々な土地を旅した経験があるらしく、外国の文化や言語、地理に関する話題には、珍しく興味を示すことがある。  
* **背景** : 第三騎士団の生え抜きの斥候。セリーヌ同様、第一騎士団への派遣任務から復帰したばかり。アバカンに息子がおり、時折、息子の話（木登りが好き、など）をぽつりと漏らすことがある。彼の過去や、斥候としての能力をどこで身につけたのかは、団員たちも詳しくは知らない。  
* **能力** :  
  * **隠密・潜入** : ミラージュ・ヴェールによる気配遮断、光学迷彩。足音を消す歩法、気配を読む能力。  
  * **飛行** : フロート・ウィングによる短距離飛行・高所への移動。  
  * **斥候技術** : 高度な追跡、偵察、地図作成、罠設置・解除（バインド・トラップ、物理罠）。  
  * **戦闘** : 鉤爪を用いた近接戦闘、暗殺術。【追記】 仲間の陽動などを利用し、敵の意識外から一撃で仕留めるのを得意とする。  
  * **特殊魔法** : エコー（音響操作、声真似）、ミラージュ・シフト（範囲撹乱）、ヴォイド・コントロール（低級生物の精神操作、眷属討伐任務でリスを操り使用）。  
  * **基礎魔法** : 自然四属性の基礎も習得している。  
* **今後の可能性** : その卓越した隠密能力と特殊な魔法により、イリアス軍の動向調査、ヴィクトルの陰謀の察知、あるいはデーモンの計画を探る上で、極めて重要な役割を担うことになるだろう。彼の過去や個人的な動機も、物語が進むにつれて明らかになるかもしれない。

## **ガルド・アイゼンヴァルト (Gald Eisenwald)**

* **役割** : ロディニア第三騎士団・団長。リエルの直接の上官。マグヌスの元弟子。  
* **年齢** : 40代後半と思われる。  
* **外見** : 壮年の男性。戦場を生き抜いてきた厳しさと、指揮官としての威厳を漂わせる。灰色の鋭い目、白髪が混じり始めた黒髪、手入れされていない無精ひげ。長身で筋肉質な体躯。  
* **服装** : 第三騎士団の団長用制服（実用性を重視したデザイン）。武器は雷の魔力を帯びたハルバード（雷槍）。防御には巨大なヒートタワーシールド（表面を赤熱させ攻撃・防御・保温、さらには調理にも使う多機能装備）を用いる。  
* **性格** : 基本的に厳格で、軍規や任務達成を重んじる実利主義者。しかし、その根底には強い責任感と部下への深い思いやりがあるが、それを素直に表に出すのが苦手な不器用な面も。心を許した相手や、酒が入った場では、意外と砕けた一面を見せることもある。\*\*\*\*【追記】隠れた芸術的才能（特に彫刻）\*\*\*\*を持っており、多忙な任務の合間を縫って、人知れず隠れ岩場で亡き妻（あるいは他のモチーフ）の像をトールズ・ボルグ（岩塊生成・削り出し魔法）で作ることが、唯一の心の慰めとなっている。「物を形作る」という行為そのものに喜びを感じる性質。家の過去の失敗（詳細は不明だが、軍事的な失策かスキャンダル？）が重圧となっており、その回復に努めている。師であるマグヌスに対しては、その力を認めつつも、その非人間性や危険性を警戒し、管理下に置こうと努めている。軍内部での政治的な立ち回りも必要に応じて行うが、本質的には現場主義者。その実直さと部下を守る姿勢から人徳があり、団員からの信頼は厚いが、背負うものが多く常に苦労が絶えない。【追記】 眷属の遺体を「貴重な研究材料」として冷静に回収させるなど、目的のためには手段を選ばない冷徹な側面も持つ。【追記】リエルの処遇を巡っては、表向きは彼女の利用価値と危険性を天秤にかけ、あくまで管理下に置くという実利的な姿勢を見せるが、内心では彼女の境遇に同情し、個人的な感情に流されすぎないよう自制している（枷）。  
* **背景** : ロディニアの名門軍人家系（ アイゼンヴァルト 家）の出身だが、過去に一族が大きな失敗を犯したため、その汚名返上が彼の代の課題となっている。セリーヌの旧領地を、彼女に代わって管理している。アバカンに息子がいる。若い頃は彫刻家を目指していたが、家の事情で軍人の道を選んだ。マグヌスの数少ない元弟子の一人であり、師が常闇の結晶果に手を出した過去を知っている。 【追記】セリーヌ・ヴァルトナーとは親戚関係にある。  
* **能力** :  
  * **卓越した指揮・戦術能力** : 特に少数精鋭部隊を率いての特殊任務や、困難な状況下での作戦遂行に長ける。  
  * **槍術** : ハルバードを用いた強力かつ精密な槍術。  
  * **魔法** : 雷属性（トール）と大地属性（ヨルズ）の魔法を得意とする。防御魔法テトラ・シールドの基礎も習得。得意技はトール・スピア（雷槍）、トールズ・ボルグ（岩塊生成・削り出し）。  
  * **ヒートタワーシールド** : 特殊な魔法が付与された盾。防御力に加え、表面を赤熱させることで攻撃（接触した敵を燃やす）、防御（熱による威嚇・溶解）、保温、調理など多目的に使用する。  
* 【追記】**現状** : セルトナの森任務（ 8章 ）で腕を負傷したが、治療中。指揮は可能。
