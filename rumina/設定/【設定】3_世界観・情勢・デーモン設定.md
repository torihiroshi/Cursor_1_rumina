# **世界観・情勢・デーモン設定（「魔法使い狩り」修正版）**

## **【世界観設定】**

**全体** : 魔法と科学、そして「魔界」と呼ばれる存在やそれに連なる力が存在するダークファンタジー世界。創造神ルミナ（太陽モチーフ）と\*\*原初の暗黒（宇宙的な根源としての闇。リエルが交信するデーモンとは異なる存在）\*\*に関する古い神話が存在する。

**【追記：古代の歴史と人類の起源】**

* **原初の混沌** : かつて、現在のイリアスやロディニアを含む地域は、常闇の眷属やデーモンのような種族が支配する地であり、彼らは常に互いに争い、世界は混沌に満ちていた。  
* **人類の祖先** : しかし、いつしかその眷属の中に、永遠に続く戦いと混沌を望まない者たちが現れた。彼らが現在の人類の直接的な祖先にあたる。  
* **第一次対デーモン戦争とルミナ教の誕生** : 人類の祖先たちは、終わらない戦いの元凶が\*\*「デーモン」——人々の恐怖や想像から生まれた歪んだ存在——\*\*にあると考え結束し、デーモンおよびそれに付き従う眷属たちとの間で大規模な戦争が勃発した。激しい戦いの末、人類の祖先はデーモンとその勢力を滅ぼし（あるいは封じ）、戦いを終結させた。その後、生き残った者たちは、二度と争いを繰り返さないようにとの願いを込めて、平和と調和、慈悲の心を説く「ルミナ教」を創始した。これは現在まで続くルミナ信仰の起源である。  
* **古代の血の継承** : かつての古代眷属の血は、形質として現在の人類にも受け継がれている。特にエリンディルの里の民のような、特定の地域や血統の者たちにはその血が色濃く残っており、これが常闇の結晶果への適性や、特異な魔法（闇魔法など）の発現に関係している。  
* **古代魔界文明の痕跡** :  
  * **遺跡** : 物語中に登場する堅牢で巨大な遺跡（リエルが訪れた山頂や丘陵地帯の遺跡など）は、人類の祖先がデーモンと戦う以前の、原初のデーモンたちが築いた古代文明の残骸である。これらは極めて強力な魔力によって建造されたため、長い年月を経ても完全には崩壊せず、当時の姿を一部留めている。  
  * **魔道具** : 同様に、古代文明で使われていた強大な力を持つ魔道具も、遺跡などに残されていることがある。リエルが 5章 で手に入れた三本爪の杖も、そうした古代の遺物の一つである。  
* **ルミナ信仰の現状** : 古代に創始されたルミナ信仰は、時代と共にその形を変え、地域によって信仰の度合いも異なり、全体的には薄れつつある。【更新】イリアスでは、かつては魔法使いの里を中心に人里離れた場所に教会が存在し、信仰が生活の一部だったが、**魔法使い狩り**によってその基盤ごと破壊された（ただし、秘境などに信仰が隠れ残っている可能性はある）。ロディニアにおいても信仰は日常生活からは薄れつつあるが、首都アバカンなど一部の地域には教会が現存し、国家行事などでその権威が利用されることもある。  
* **【追記：未来の呼称（裏設定）】** : 物語の舞台となるイリアス・ロディニアを含む周辺地域は、後の世において「プロゲニア (Progenia)」と呼ばれるようになる可能性がある。これは「進歩 (Progress)」と「起源 (Genesis)」を組み合わせた造語であり、新たな時代の始まりや、変化・進化していく土地といったニュアンスを持つ（※本文中には登場しない可能性のある背景設定）。

**気候・地理** :

* **ロディニア** : 北方に位置し、首都アバカン周辺は厳しい寒さと永久凍土に覆われている。資源は乏しい。  
* **イリアス** : ロディニアの南に位置する国。比較的温暖で資源も豊富？（**長銃**の原料など）。魔法使いの里（エリンディルなど）が点在していた。  
* **その他** : 外国には、実体を持つデーモンが出現する地域が存在するらしい。

**文明レベル・技術** :

* **ロディニア** : 凍土という厳しい環境を克服するため、魔法技術が生活の隅々（建物保温、洗濯槽、浮遊リフト、ナリチャボなど）まで浸透し、実用的・インフラ的に高度に発展している。軍事力も魔法が中心（魔法騎士団、魔法使い団）。ラタノフ（生活魔法）、マグヌス（戦闘・理論魔法）のような天才的研究者が技術革新を支えているが、一般市民の学力レベルや基礎的な工業技術（製鉄など）はイリアスに劣る可能性がある。【追記】 魔法の発展方向性が、エリンディルのような自然共存・儀式的なものとは異なり、より体系的・技術的である。  
* **イリアス** : 魔法を（表向きは）捨て、科学技術と火薬兵器を発展させている。**長銃**を開発・配備し、魔法障壁をも貫通する脅威となっている。一方で、裏では「魔剣部隊」のように魔界の力（あるいはそれに類する技術）を利用している疑いがある。【追記】 海の向こうの国々との貿易も行っており、航海技術や海洋学など、国外に意識を向けた学問も発展している。 【追記】水面下では、ヴィクトルらが関与する形で、旧来の魔法とは異なる独自の「技術魔法」（例：鉄魔法アダマス）の研究開発が進められている。これは工業化と親和性が高く、精神性への依存を排した体系を目指している。

**【New】ロディニアの特異自然現象：「雪の波」** :

* **発生場所** : 主にロディニア国内に点在する 特定の地質構造を持つ地域 で観測される特異な自然現象。 古代文明の遺跡が近くに存在することが多い。  
* **現象** : まるで海が荒れるように、 広大な雪原が柔らかく波のように隆起し、うねり、時には雪崩のように崩れ落ちる 。発生は不定期かつ局所적だが、規模が大きい場合は周辺地域に甚大な被害をもたらすこともある。 常人には聞こえないが、特定の技術（例：イリアスの技術魔法）や能力（例：フィンの音魔法）を持つ者には、地下の岩盤が発する特有の周波数が感知できる。魔法力を感知できる者にとっては魔力が濃く感じられる場所であり、自然魔法に近いが異なる性質を持つ。波は常に発生し続けており、通常の歩行や（もし水があれば）泳ぎでの移動は困難を極める。  
* **原因（仮説）** : \*\*地下深部の特殊な岩盤活動が主因と考えられている。\*\*ただし、近くにある古代遺跡から放たれる微弱だが特殊な魔力が、この岩盤活動と共鳴し、現象を増幅・変質させている可能性も指摘されている。ロディニアでは地質学的な研究が進んでいないため、 一般的には遺跡の魔力や、あるいは古代の呪いのようなものが主な原因だと信じられている ことが多い。  
* **【New】生態系への影響** : この特異な環境に適応した独自の生物が存在する。雪の波間を泳ぐように移動するアザラシに似た生物や、波によって掘り起こされた地中の生物を捕食する特殊な魚類などが見られる。また、常闇の結晶果とは関係なく、雪の波がもたらす特殊な魔力に適応した独自の進化を遂げた生物もおり、翼を持たない竜のような姿をした「スノウドレイク」が稀に観測される。植物に関しても、エリンディルのように濃密な魔力を帯びた環境であるため、「雪の波」周辺では、雪の上に直接根を張り、紫色に淡く輝く氷のような花（通称：氷晶花）など、特殊な植物が存在する。  
* **【New】遺跡との関連** : 遺跡自体が雪の波の直接的な原因ではないが、その存在が現象に影響を与えていることは確かである。遺跡の魔力が波と共鳴することで、予測不能な魔力異常（一時的な魔力フィールドの発生や消失、空間の歪みなど）を引き起こすことがあるとされる。

**【追記：ロディニアの貴族制度と役割】**

* **権力分立** : ロディニアでは、軍事（騎士団）、政治（行政）、魔法（魔法使い団）の権力が比較的明確に分離されている傾向がある。貴族はその出自や役割によって、これらのいずれか、あるいは複数に関与する。  
* **王家（Royal Lineage）** :  
  * 起源: 魔法によってロディニアの礎を築いたとされる最初の王の直系の子孫。  
  * 権限: 現在の王は、軍の最高指揮権は持たないが、行政における最高決定権を持ち、騎士の叙任権なども有する。象徴的な意味合いも強い。  
* **軍系貴族（Military Nobility）** :  
  * 起源: 戦功や領土防衛への貢献によって王から叙任された軍人の家系。最も多くの貴族がこのカテゴリーに属する。  
  * 役割: 各騎士団の中核を成し、領地の統治や防衛を担う。最高位は各騎士団長（第一、第二、第三の順に序列があり、管轄する領土の規模もこれに準じることが多い）。カイルのベラルド家などがこれにあたる。  
* **デーモンハンター貴族（Demon Hunter Nobility）** :  
  * 起源: デーモンや常闇の眷属の討伐といった、国家への特別な貢献によって叙任された家系。エレナのラムスドルフ家が代表例。  
  * 現状: 軍組織とは直接的な繋がりが薄い場合が多く、派閥としての規模も小さい。世襲している領地も広大とは言えず、現在は「名誉貴族」のような側面が強い家もある。  
* **魔法使い貴族（Magician Nobility）** :  
  * 起源: 魔法が広く普及するロディニアの中でも、特に傑出した魔法能力を持ち、その技術や知識によって国家に多大な貢献（生活魔法の開発、国防への技術協力など）をした人物が、一代限りの、あるいは世襲の貴族として叙任されるケース。  
  * 役割: 首都の魔法研究所や、地方の魔法関連施設・土地の管理運営などに関与することが多い。マグヌスやラタノフがこの例。生活魔法の開発などで民衆の暮らしに貢献しているため、他の貴族派閥に比べて民衆からの親しみは持たれやすい。  
  * 備考 : ヴィクトルは強力な魔法使いだが、現時点では国家への明確な「貢献」が少ない（あるいは隠している）ため、貴族ではない。

**【追記】**常闇の結晶果**（とこやみのけっしょうか）**

* **発生源** : 大量の死（戦争、虐殺、災害など）があった場所に、死者の無念や負の感情、あるいは流された生命力を吸って、捻じくれた枯れ木のような植物に実る、赤黒いザクロに似た禁断の果実。  
* **効果** : 生物が持つ潜在的な「魔の才能」や、特殊な血筋（例：エリンディルの「魔界の血」＝古代眷属の血）を強制的に覚醒・増幅させる触媒。食べた者の精神や肉体に劇的な変化（超感覚、闇魔法の発現、外見の変化、味覚喪失など）をもたらす。効果には個体差が非常に大きい。  
* **危険性・副作用** : 強大な力を与える代償として、精神汚染のリスクが極めて高い。特にデスティールのような魂に干渉する力を得た場合、その使用衝動や快感に抗うことが困難になり、理性を失い、破滅に至る可能性がある。適合しない者が食べると、精神崩壊や肉体の崩壊を引き起こし、死に至る（過去にロディニア騎士が誤食し錯乱死した例あり）。  
* **エリンディルでの扱い** : 18歳の成人の儀式において、希望者が世界の調和の「楔」となる覚悟をもって食べる、という形で管理・利用されていた（ただし、その力の危険性も認識・警戒されていた）。  
* **デーモンの利用** : デーモンは、自身の目的（混沌の拡大、リエルの力の解放）のために、結晶果の生成を促し、リエルに強制的に食べさせた。  
* **眷属化との関連** : 魔法使いの血筋（古代眷属の血）を持たない動物などが偶然食べた場合、強大な力を得るものの理性は失われ、「常闇の眷属」へと変貌する原因の一つと考えられている（魔狼など）。ただし、デーモン曰く、これらは「同類」ではない。  
* **【New】魔力隠蔽** : 結晶果自体が魔力を隠蔽する能力を持つ可能性があり、発見が困難な場合がある（ 9章 ヴィクトルの発言より）。

## **【デーモン設定】**

**リエルが交信するデーモン** :

* **性質** : 実体を持たない、世界の魔法そのもの、あるいは人々の集合的無意識（特に恐怖や神話への信仰）が生み出した産物。原初の「暗黒」そのものではなく、それに基づく人間の想像や恐怖が形作った存在に近い。  
* **【追記】外見（リエル視点）** : 黒曜石のような肌、山羊角、皮膜の翼、地面に届く長い腕、鋭い爪、無数の紅い目。爪からは粘液のようなものが滴る。  
* **目的** : 自身の存在基盤である「忘れられること」への恐怖から、世界に混沌をもたらし、自身の存在感を刻みつけ、影響力を維持・拡大すること。その手段として、戦争を長引かせ、死を増やすことで常闇の結晶果を生成・拡散させ、自身と繋がる存在（＝自身を認識し、恐れる存在）を増やそうとしている。【更新】特にリエルのような「魔界の血」を持つ者を目覚めさせ、その潜在能力を解放し、自身の駒として利用することにも強い関心を持つ。  
* **能力** : 精神交信（結晶果を通じた繋がり）、広範な情報収集能力、リエルへの影響力（誘惑、精神攻撃）、物理現象への限定的な干渉（リエルの周囲の闇の発生など）。物理的な戦闘力は未知数（本人は強力だと言っているが）。【追記】竜の眷属を召喚・使役する能力を持つ。 【追記】聖剣**スノウタリア**を介してエレナにも干渉可能（9章）。  
* **弱点** : 魔除けのネックレス（精神交信遮断）、聖剣**スノウタリア**（魔界の力への特攻、 ただしデーモンはこの剣の破壊を目論んでいる ）。人々の信仰や恐怖が薄れれば、存在自体が希薄になる可能性。

**原初の「暗黒」** : ルミナ神話に登場する存在。宇宙規模の根源的な闇。リエルが交信するデーモンとは別物で、もしかしたらルミナ（太陽）とは敵対せず、共存あるいは中立的な関係（太陽と宇宙のような）にあるのかもしれない。詳細は不明。 【追記】11章でリエルに直接語り掛け、彼女の選択が世界の調和を左右することを示唆する。

**実体を持つデーモン** : かつて（20年前など）ロディニアにも出現し、エレナの父（パーブル）が討伐したとされる。現在は主に外国に出現するとされる。パーブルはこれを追って放浪している。リエルが遭遇したデーモン（およびその眷属）との関係は不明。

**【追記：デーモンの種類と思考・行動指針の違い】**

* **根源的な動機** : 世界に登場するデーモン（あるいはそれに類する存在）は、個体差はあれど「忘れられることへの恐怖」を根源に持ち、「混沌をもたらす」ことで自身の存在を世界に刻みつけようとする点で共通している。しかし、その手段や思考様式には違いが見られる。  
* **イリアスのデーモン（リエルが交信する存在）** :  
  * **戦略** : 自身の完全復活（あるいはより強大な存在への変貌？）を目指し、その手段として、人間同士の争いを煽り、死を増やすことで常闇の結晶果を生成・拡散させる。結晶果や自身の影、眷属の干渉を通じて、世界そのものを徐々に魔界（かつての古代魔界文明のような環境）に近づけ、最終的に自身が顕現しやすい環境を整えようとしている。  
  * **特性** : 実体を持たない精神的存在だが、門を使わずとも短時間であれば限定的ながら地上に影響を及ぼすことが可能。狡猾で、人間心理を巧みに操り、効率的に混沌を広げようとする。  
* **過去のデーモン（20年前の事件）** :  
  * **戦略** : 特定の魔法使いの里（イリアス国内）に干渉し、彼らに「大儀式」を行わせることで、無理やり現世への「門」を開かせ、実体を持って顕現した。イリアスのデーモンのように人間同士を争わせて効率的に世界を魔界化するのではなく、自身の強大な力と恐怖によって直接的に世界を支配・変貌させようとした。  
  * **結果** : デーモンハンター（パーブル含む）や当時の騎士団などによって最終的に討伐された。しかし、この事件はイリアスに大きな爪痕を残した。  
  * **影響** : この事件において、魔法使いの里がデーモンを呼び出す儀式を行った（あるいは、そう見なされた）ことが決定的な要因となり、イリアス国全体が魔法使いの里に対して強い不信感と憎悪を抱くようになった。これが後の**魔法使い狩り**に繋がる大きな原因の一つとなる。 【追記】このデーモンは各地で人々に無差別に力を与え、混沌を広げようとしていた（9章エレナの発言より）。

**【New】デーモンの古文書の変容と歴史的背景**

1. **古文書の起源とエリンディルの役割**  
   * 『デーモンの古文書』は、その名に反して、元々は単なる邪悪な魔導書ではなかった。その起源は、かつて人類の祖先が「デーモン」と呼ばれる存在と戦った古代にまで遡る。  
   * **世界のバランサーの書**: エリンディルの里の伝承や、原初の闇が示唆するところによれば、この古文書は本来、世界の光と闇の調和を保つ「世界のバランサー」としての役割を担う者が、その力を正しく理解し、行使するために存在したとされる。  
   * **心の在り方による解読**: 古文書の真の解放は、力や知識だけでなく、それを扱う者の「心」——特に世界の調和を願う清浄な意志や、他者への慈しみ——によって進むとされていた。エリンディルの民が、常闇の結晶果を成人の儀式で食し、自らを「ルミナ様の傍らにいる暗黒」と任じて世界のバランスを保とうとしたのも、この古文書の本来の教えに基づいていた可能性がある。  
2. **デーモンによる汚染とリエルの古文書**  
   * しかし、リエルが現在手にしている焼け焦げた古文書は、イリアスに干渉するデーモンによって、その性質を大きく歪められている。  
   * **デーモンの道具**: リエルが交信するデーモンは、自身の存在を世界に刻みつけ、混沌を拡大するために、この古文書を利用しようとしている。彼は、古文書を完全に解読・掌握することで、自身がより強大な力を持って現世に顕現するための「門」として利用する計画を持つ。  
   * **歪められた解読条件**: デーモンは、古文書の解読条件を「魂の吸収（デスティール）」という、より直接的で禍々しいものへと歪めた。これにより、古文書の力は本来の調和のためのものではなく、破壊や憎悪を増幅させる方向へと捻じ曲げられている。リエルがデスティールを使うたびに新たな知識（闇魔法）が流れ込んでくるのは、このためである。  
   * **リエルの血筋**: リエルの「魔界の血（古代眷属の血）」は、この歪められた古文書と強く共鳴し、デーモンの干渉を受けやすい状態にある。  
3. **古文書の変容（11章以降の展開）**  
   * 物語が進む中で、リエルの古文書には大きな転機が訪れる。  
   * **原初の闇の声とエレナの影響**: 10章でリエルがエレナとの倫理的な対話を経て、さらに原初の闇（宇宙的な調和を司る存在）からの問いかけを受けることで、彼女の心に変化が生じる。この内面的な変化と、エレナという「光」の存在（リエルにとってのルミナ）との絆が深まることが触媒となり、古文書が本来の姿を取り戻し始める。  
   * **闇魔法の喪失と新たな力の覚醒**:  
     * デーモンの影響が薄れることで、これまでリエルが使えていた闇魔法（オーブネス、シャドウ・バインド、デスティール、強化イグニスなど、デーモンの干渉によって歪められた力）が、以前のようにうまく使えなくなる。  
     * 代わりに、古文書のこれまで空白だったページに、新たな文字や図形が浮かび上がり、本来の「調和のための力」が目覚め始める。  
     * ステラ・オーブ（光・闇）: 眷属を浄化し、闇が浅い者を元に戻す可能性を秘めたエネルギー球。  
     * スピリトゥス・アストラム（光・闇）: ウサギのような精霊を召喚し、偵察や伝令、限定的な治癒、さらには戦闘補助（巨大化して剣を咥えて戦うなど）も可能な、かつてのシャドウ・バインドに代わる力。  
     * トワイライト・パージ（光・闇）: 救済不能な魂を吸収し、原初の闇へと還すことで古文書の真の解放を進める、デスティールとは異なる性質の魂への干渉魔法。  
   * **古文書の真の役割の理解**: リエルは、この変容を通じて、古文書が本来、エリンディルの民が「世界のバランサー」として、強い意志と調和を願う心によって開かれるべきものであったこと、そしてその力が復讐のためではなく、世界を守るために存在することを知ることになる。  
4. **ナベリウスと別の古文書（今後の展開）**  
   * 物語の後半では、吸血鬼ナベリウスが、リエルの持つものとは異なる（あるいはその一部である可能性も）『デーモンの古文書』を完成させ、イリアスの地にかつて封印されていたデーモンを完全復活させるという展開が予定されている。  
   * **古文書の多様性**: このことは、デーモンの古文書には複数のバージョンが存在する可能性や、あるいは解読の段階や方法によって、異なる結果（異なるデーモンの復活や、異なる力の顕現）をもたらす可能性を示唆する。  
   * **デーモン復活の触媒**: ナベリウスは、イリアス各地に隠れ潜んでいた魔法使いの生き残りを虐殺し、その質の高い魔力を吸収することで古文書を完成させるとされる。これもまた、デーモンが人間の魂や魔力を糧として力を増し、古文書を介して現世に影響を及ぼそうとするパターンの一つと言えるだろう。

このように、デーモンの古文書は、物語を通じてその姿と役割を大きく変えていく、非常に重要なアイテムとなる。リエルの成長と心の変化、そして世界の運命と深く結びついた存在として描かれていく。

**【New】「デーモン」と「才能」、そして「エレナ（ルミナ）の存在」について**

* 「人間型デーモン」という捉え方:  
  物語の世界において、リエルやナベリウスのような存在は、従来の「デーモン」とは異なる、いわば「人間型のデーモン」として捉えることができる。  
  これは、プロゲニア（物語の舞台となる地域）の人々の祖先が古代の眷属であったという設定に基づき、現代人の中にもデーモン化しうる潜在的な「才能」を持つ者がいることを示唆する。  
* ヴィクトルの目指す世界との関係:  
  ヴィクトルが目指す「新しい時代」とは、このような生まれ持った「才能」や血筋によって運命が左右されることのない、真に平等な世界の可能性がある。彼は、絶対的な「才能」に人々が振り回される現状を打破しようとしているのかもしれない。  
* リエルとナベリウスの分岐点:  
  リエルとナベリウスは、共に「人間型のデーモン」とも言える素養を持ちながら、異なる道を歩むことになる。その運命を分けた最大の要因は、「エレナ（ルミナ）のような存在が近くにいたかどうか」という点に集約される。  
  エレナの存在は、リエルにとっての「光」や「導き」となり、彼女が完全な闇に堕ちることを防ぎ、「調和のための力」に目覚めるきっかけを与える。一方、そのような存在がいなかったナベリウスは、より純粋な形でデーモンの力に染まっていくのかもしれない。これは、個人の資質だけでなく、周囲の環境や人間関係が、その人の生き方や運命に大きな影響を与えることを示す。

## **【イリアス/ロディニア情勢】**

**戦争状態** : 現在、両国は国境付近で散発的な戦闘（小競り合い）が続く\*\*「緊張状態」\*\*にあり、いつ本格的な全面戦争に突入してもおかしくない。

**開戦理由（イリアスの主張）** : イリアス国内でのデーモン出現を、魔法国家ロディニアによる侵略行為と断定。魔法そのものが世界の理を歪め、いずれ不死の魔法のような破滅的な力に至る危険なものであるとし、\*\*「魔法の撤廃」\*\*を大義名分としている。

**【New】イリアスにおける「過激派」魔法使いの連合**

* **かつての存在**: イリアスには、エリンディルのような穏健な魔法使いの里だけでなく、より強力な力、特に闇の魔法に傾倒し、その探求のためには手段を選ばない「過激派」とも言える魔法使いたちの連合が、かつて（あるいは**魔法使い狩り**直前まで）存在していた。彼らは、力の追求を至上とし、時には禁忌とされる古代の魔術や、デーモンとの交信すら試みていたとされる。  
* **ナベリウスの出自**: 吸血鬼ナベリウスは、この「過激派」魔法使いの連合の生き残りであり、その中でも特に古代の「魔界文明の血」を色濃く引く特異な存在である。彼が持つデーモンへの強い信奉心や、力の追求への渇望は、この連合の思想的影響を強く受けている可能性がある。  
* **魔法使い狩りとの関連**: このような「過激派」の存在や、彼らが行っていたとされる危険な研究・儀式が、イリアス全体の魔法使いへの不信感を増幅させ、**魔法使い狩り**をより過激化させる一因となった可能性も考えられる。彼らの活動が、イリアス政府にとって「魔法＝悪」というプロパガンダを強化する格好の材料となったのかもしれない。

**魔法使い狩り** : 上記（開戦理由）を大義名分に、イリアス国内の魔法使いの里（エリンディル含む）を、デーモンやロディニアとの内通の疑いをかけて殲滅した。【追記】 20年前に起きた「実体を持つデーモン」の出現事件において、魔法使いの里がデーモンを呼び出す儀式を行った（あるいは、そう見なされた）ことが、イリアス全体の魔法使いへの強い不信感と憎悪の根源となっており、**魔法使い狩り**を正当化する大きな要因となっている。

**【更新】襲撃部隊** : 兵士たちは魔法使いをデーモンと同一視し、殲滅を正義と信じる狂信的な側面を持つ。彼らが口にする「20年前の二の舞」とは、過去に実体を持つデーモンが出現し甚大な被害をもたらした事件を指しており、これが兵士たちの強い動機付けやトラウマとなっている。エリンディル襲撃時には、強力な結界を突破し、さらに\*\*里内部で魔法（浮遊魔法など）を阻害・無力化する何らかの手段（装置や特殊技術）\*\*を使用した。これはヴィクトルの暗躍を示唆している。

**ロディニアの立場** : デーモン出現はイリアスの言いがかりだと反発。自国も常闇の眷属の被害に悩まされており、デーモンを操っているわけではないと主張。イリアスの侵攻に備え、国境付近に第一・第二騎士団を主力として配備。魔法技術のさらなる向上（対**長銃**、対魔剣部隊）を急いでいる。

**魔剣部隊** : イリアス軍が投入してきた謎の部隊。常闇の眷属が使うような、魔力を帯びたいびつな形状の剣（魔剣）を使用し、高い戦闘力を持つ。その武器の出所や、魔界との関連がロディニア側の大きな懸念事項となっている。 【追記】常闇の眷属の斥候が戦場を監視しており、デーモンが魔剣部隊に関与している可能性が濃厚（9章セリーヌの報告より）。また、ロディニア兵の間で魔剣部隊への憎悪が広がっている（9章セリーヌの発言より）。

**ロディニア軍内部** : 【更新】

* **騎士団** :  
  * **第一騎士団** : 最大規模の兵力を有し、主に大規模作戦の指揮や正面戦力を担当。伝統と規律を重んじる保守的な気風が強いとされる（特に副官など）。【追記】団長自身は物語開始時点でヴィクトルの思想に共鳴しており（あるいは既にヴィクトル派の一員であり）、その影響で第一騎士団内部の一部もヴィクトルの掲げる「変革」に傾倒しつつある。これは保守的な副官との間に内部的な緊張を生んでいる。  
  * **第二騎士団** : 魔法戦闘に特化した部隊。所属する騎士や魔導士は高い魔法能力を持つ。魔法の専門家として、未知の魔法（リエルの闇魔法など）に対しては強い関心と同時に警戒心も抱いている。

    * 憲兵隊（ガルド直属?） : 首都防衛の要であり、個々の隊員が騎士クラスの実力を持つとされるが、精鋭故に任務で動かすことは難しい。

    * 弓兵隊（セリーヌ隊） : セリーヌが隊長。イリアス、ロディニア、他国出身者や無国籍者など、様々な背景を持つ「ならず者」が集まっている。戦争を楽しむ者が多いが、最低限の規律は守られている。なぜか半裸のマッチョが多く、弓兵でありながら素手での殴り合いを好む者もいる。  
    * 斥候隊（フィン隊） : フィンが隊長を務める謎多き部隊。隊員にはロディニアやイリアスとは異なる人種と思われる者も多い。ロディニアの自然魔法とは異なる「音魔法」の使い手たちが所属している。  
  * **共通** : 騎士は基本的に貴族階級出身者が多いが、実力主義的な側面もある。
