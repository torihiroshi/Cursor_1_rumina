# **物語のプロット案（第一部）**

## **【物語の前提（9章完了時点）】**

リエルは第三騎士団に所属し、約二ヶ月間の訓練と最初の眷属討伐任務（セルトナの森）を経験。仲間（特にカイル、エレナ）との絆を深めつつあるが、自身の闇の力（デスティールの衝動）と過去の罪（5章での父娘襲撃）に深く葛藤している。エレナはリエルの力とデーモンの気配に強い警戒心を抱き始めており（7章ラストの大浴場シーン、8章の戦闘など）、二人の間には表面的な穏やかさの下に緊張感が存在する。

ロディニアとイリアスは国境で小競り合いを続ける\*\*「緊張状態」\*\*。本格的な全面戦争の勃発が間近に迫っている。

デーモンはリエルへの精神的干渉を続け、彼女の復讐心を煽り、闇の力をさらに解放させようとしている。また、戦争の永続化（＝死の増加による結晶果の生成）を目論んでいる。

ヴィクトルはロディニア魔法使い団の穏やかな助手としてマグヌスに仕えつつ、裏では人類の「停滞」打破と「進化」のため、イリアスと共謀し、ロディニア（特にマグヌスと魔法体制）への破壊工作を計画中。エリンディル滅亡時の魔法阻害にも関与した可能性が極めて高い。

第三騎士団は、セルトナの森任務（8章）で団長ガルドが負傷（腕）、隊員一名が犠牲となったが、任務は達成。次なる大きな任務として、イリアス領内の拠点「アルヌス」への攻略作戦を控えている。

## **【9章～11章】**

1. **エレナの内省と決意** : エレナがリエルの異変（7章の大浴場での瞳の発光、8章の戦闘でのデスティール抑制時の**スノウタリア**の反応など）、デーモンの干渉の危険性、父パーブルの過去（デーモンハンターとしての戦い）、そして自身の家系と宿命について深く思い悩む。リエルを個人として守りたい気持ちと、彼女の力がもたらすかもしれない脅威への警戒心との間で葛藤するが、最終的にリエルをデーモンの影響から守り、その力を（可能ならば）制御下に置くことを決意する。  
2. **緊急会議** : エレナ（及び、リエルの異変を目撃した可能性のある他の者）が、ガルド団長とマグヌスにリエルの状態とデーモンの干渉について詳細を報告。リエルへの直接的な精神干渉、物理現象（オーラなど）への影響、そして聖剣**スノウタリア**がリエルの闇の力やデーモンの気配に強く反応することが共有され、緊急の対策会議が開かれる。ヴィクトルも同席し、冷静に意見を述べる（裏では情報を収集）。  
3. **スノウタリア強化とネックレス譲渡** : 会議の結果、対デーモン・対闇魔法の切り札として、エレナの**スノウタリア**を強化することが決定。魔法研究所にて、マグヌス主導の下、ヴィクトル、そして引退していた生活魔法の権威ラタノフ（登場）が協力し、特殊な魔石を装着・調整する作業が行われる。これにより聖光刃の威力・安定性が向上（ただし使用制限は残る）。作業後、エレナはリエルの精神を守るため、そして信頼の証として、父から譲り受けた魔除けの銀のネックレスをリエルに譲り渡す。リエルは戸惑いながらも受け取る。  
4. **束の間の日常** : **スノウタリア**強化完了後、アルヌス作戦開始までの数日間。リエルはネックレスの効果か、あるいは自身の決意か、一時的にデーモンの声から解放され、穏やかな時間を過ごす。訓練場でのカイル（剣術指南）、セリーヌ（弓や体術の自慢）、フィン（珍しく外国の話をする？）との交流。夜には騎士団御用達の酒場で、ガルド（腕は治療中だが快方）、マグヌス（研究の合間に渋々参加？）、エレナ、カイル、セリーヌ、フィン、そしてリエルも交えた、ささやかな宴会が開かれる。ガルドの意外な一面（酒が入ると饒舌になる、彫刻の話を少し漏らす？）や、マグヌスの人間味のなさ（あるいは戦闘や魔法の話になると饒舌になる？）、仲間たちの普段見せない表情などが描かれ、リエルはアバカンでの居場所に複雑ながらも確かな温かさを感じる。  
5. **ナベリウス襲撃** : 宴会が終わり、リエルたちが酒場から出た直後、あるいはその翌日の夜などに事件発生。アバカン市街（あるいは騎士団本部の近く）に、強力な常闇の眷属である吸血鬼ナベリウスが出現し、無差別に市民や衛兵を襲い始める。異変を察知したカイルが真っ先に駆けつけ、追跡・応戦を開始。すぐに第三騎士団の緊急出動となる。  
6. **激闘と暴露** : 騎士団本部の中庭、あるいは近くの広場などで、ナベリウスと第三騎士団（リエル、エレナ、カイル、セリーヌ、フィン、他）との激しい戦闘が展開される。ナベリウスは高度な闇魔法（シャドウ・ランス、アビサル・ファング・ブルーム等）、驚異的な身体能力と再生能力で騎士団を圧倒する。戦闘の最中、ナベリウスはリエルの「魔界の血」の匂いを嗅ぎつけ、彼女が過去に犯した罪（ 5章 での父娘襲撃、特に少女へのデスティール）を、本人にしか分からない形で、あるいは挑発的に暴露する。リエルは激しく動揺し、罪悪感と自己嫌悪に苛まれ、精神的に打ちのめされる。  
7. **救援と頂上決戦** : リエルがナベリウスに追い詰められた瞬間、エレナが強化された聖光刃を放ち、リエルを救出する。その光はナベリウスに確かなダメージを与える。そこへ、報告を受けて駆けつけたガルド、マグヌス、そしてヴィクトルを含む魔法使い団の援軍が到着。数の不利を悟ったナベリウスは、他の騎士たちを牽制しつつ、マグヌスを指名。「貴様とは決着をつけねばなるまい」と告げ、二人はフロート・ウィングでアバカンの夜空高くへと舞い上がり、一対一の決戦を開始する。  
8. **マグヌス vs ナベリウス** : フィンがミラージュ・ヴェールで隠れながら、二人の壮絶な空中戦を遠距離から監視・報告する。マグヌスの圧倒的な四大元素魔法と、ナベリウスの多彩な闇魔法・再生能力が激突する。戦闘中、ナベリウスがマグヌスも過去に常闇の結晶果を口にした事実を暴露する（あるいはマグヌス自身が認める）。激闘の末、マグヌスは杖と右腕を失うほどの深手を負うが、最後の力を振り絞り、アクア・セヴァランスでナベリウスの右腕も切断する。  
9. **撤退と後遺症** : 決定打を与えられず、魔力切れを起こしたマグヌスは夜空から落下するが、フィンによって寸前で救助される。一方、右腕を失ったナベリウスも消耗が激しく、どこからともなく現れた謎の眷属（コウモリ型？あるいは別の配下？）に助けられ、夜の闇へと撤退する。「ロディニアの魔法の実力、大体わかったぞ…次はないと思え」といった言葉を残して。マグヌスは一命を取り留めるも、右腕と杖を失い、魔力的にも大きな後遺症を負う可能性。  
10. **倫理的対話**: 事件後、自身の罪を暴露され、仲間を危険に晒したことに深く落ち込むリエルが、エレナの部屋を訪れ、全て（父娘襲撃、デスティールの衝動）を告白し、「私のような者は生きていてはいけない」と死を願う。エレナは、リエルの告白と、彼女から放たれる闇の気配、そして自身のトラウマ（父の失踪、デーモンへの憎しみ）との間で激しく葛藤する。一瞬、リエルの首に手をかけかけるほどの衝動に駆られるが、寸前で思いとどまる。そして、「死んで償えると思うな。生きろ。生きて、その罪を背負い、それでも人を助け、デーモンを倒し、世界をより良くするためにその力を使え。それがお前の、そして私の戦いだ」と、厳しいながらもリエルを生かす道を示す。二人は、生きること、罪を背負うこと、そしてこれから共に歩む道について、静かに語り合う。エレナはリエルを完全に許したわけではないが、共に戦い、見届ける覚悟を決める。  
11. 【New】原初の闇の声と古文書の変容: エレナとの対話の後、一人になったリエルの意識の奥底に、これまでとは質の異なる、深く、静かで、宇宙的な広がりを感じさせる声が響く。それはデーモンのような嘲りや誘惑ではなく、ただ純粋な問いかけのようだった。『汝、闇の子よ。光は遍く存在し、闇もまた然り。ルミナは常に汝の傍らにあり、我もまた、世界の深淵にて汝を見守っている。汝が歩む道は、復讐による更なる混沌か、それとも、光と闇を受け入れ、調和をもたらす道か。汝の選択が、世界の揺らぎを決めるだろう。』声はそれだけを告げると、深淵の静寂へと消えていく。リエルは、その声が何なのか理解できないが、自身の存在と力の意味について、新たな、そしてより根源的な問いを突き付けられた感覚に包まれる。  
    【追記：リエルとデーモンの関係性の変化】この「原初の闇」との接触、そしてエレナとの絆の深化、さらにエレナから譲り受けたネックレスの効果も相まって、リエルと、これまで彼女に干渉してきた「イリアスのデーモン」との直接的な囁きや唆しは絶たれる。これにより、リエルはデーモンの完全な支配からは解放されるが、彼女の力の根源や古文書の謎は、依然としてデーモン（あるいはより根源的な闇）との繋がりを示唆しており、物語の後半で再び彼らと対峙する可能性を残す。この変化は、リエルが自らの意志で力と向き合い、世界の調和のための役割に目覚めていく上での重要な転換点となる。  
    【New】その声の影響か、あるいはエレナという存在（リエルにとってのルミナ）との繋がりが深まったためか、リエルが持つ焼け焦げた古文書に変化が訪れる。これまでデーモンの干渉によって歪められ、デスティールによって解読が進んでいたページが輝きを失い、代わりに、本来あるべき清浄な魔力が古文書全体から微かに発せられ始める。リエルは、これまで使えていた闇魔法のうち、\*\*オーブネス、シャドウ・バインド、そしてデスティールといったデーモンの影響が強かった主要な闇魔法が完全に使えなくなり、強化イグニスも通常の炎の性質に戻ってしまっていることに気づく。\*\*同時に、古文書の別のページ——これまで空白だった、あるいは読めなかった部分——に、新たな文字や図形が浮かび上がり始めるのを見る。それは、闇の力でありながら、破壊や憎悪ではなく、世界のバランスを保つための力、調和のための魔法を示唆しているようだった。リエルは、この古文書が本来、エリンディルの民が「世界のバランサー」として、強い意志と調和を願う心によって開かれるべきものであったこと、そしてその力が復讐のためではなく、世界を守るために存在することを知る。

## **【 12章 以降：アルヌス攻略作戦と第一部クライマックス】**

1. **アルヌス作戦ブリーフィング** : ナベリウス襲撃事件により首都の警備体制が見直され、作戦は一時延期された後、改めてブリーフィングが行われる。ガルド団長（腕は治療中で戦闘には直接参加できないが、指揮は執る）が最終的な作戦計画、各部隊（カイル隊、セリーヌ隊、フィン、リエルなど）の役割分担、潜入経路、特に警戒すべき点（長銃、魔剣部隊、そしてエリンディルで使用された可能性のある魔法阻害技術への対策）を説明する。フィンからは、ロディニア軍内部の意見対立（早期決戦派 vs 慎重派）や、ガルド団長と連携する方面司令官（ヴァレリウス司令など）の動きに関する情報も報告される。リエルには、マグヌス（あるいはヴィクトル？）から、対イリアス兵器用に調整された新しい魔法武器（杖？短剣？詳細未定）が与えられる。 リエルは、以前の闇魔法が使えなくなったことに戸惑いつつも、古文書に現れた新たな力の手がかりと、仲間を守りたいという思いを胸に、作戦への参加を決意する。  
2. **出撃準備** : 部隊はアルヌスへ向け、出撃準備を進める。リエルは新しい武器と、エレナからもらった魔除けのネックレスをしっかりと身に着け、仲間と共に戦う決意を新たにする。カイルやエレナ、他の仲間たちとの、作戦前の短い会話（互いを励ます、作戦を確認するなど）。  
3. **国境越えと潜入** : フィンの先導により、夜陰に乗じてイリアス領内へと秘密裏に潜入する。国境警備を掻い潜る緊張感のある移動。初めて見るイリアスの風景（ロディニアとは異なる植生、気候、集落の様子など）をリエルの視点から描写する。  
4. **アルヌス拠点への接近・偵察** : 目標であるアルヌス補給拠点に接近。フィンによる最終偵察が行われ、拠点の詳細な防御体制（見張り塔、警備兵の配置、魔法的な防御の有無）、兵力（長銃部隊の規模、魔剣部隊の存在確認）などが判明する。  
5. **潜入開始** : 夜間、あるいは悪天候などに乗じて拠点への潜入を開始。各員のスキル（フィンの隠密・罠解除、セリーヌの遠距離からの監視・狙撃による援護、カイル隊の陽動、リエルの新たな力によるサポートなど）を活かした、サスペンスフルな展開。  
6. **戦闘勃発** : 潜入中に罠にかかる、あるいは警備兵に見つかるなどして警報が鳴り響き、拠点内でイリアス軍との本格的な戦闘が始まる。長銃による激しい銃撃、通常兵士との白兵戦、そして遂に姿を現す魔剣部隊との交戦。  
7. **アルヌス攻略戦** : 拠点内部での激しい戦闘。補給物資の破壊、指揮官の無力化、魔剣部隊との死闘、重要情報の入手（イリアスの次の作戦、魔剣部隊の秘密、ヴィクトルの関与を示す証拠など）といった目標達成を目指す。仲間との連携、犠牲、リエルの戦闘における成長と 新たな（本来の）古文書の力の覚醒・活用 （あるいはまだうまく使えない葛藤）。  
8. **首都攻防戦** : （アルヌス攻略作戦の後、戦局が悪化、あるいはヴィクトルの計画が最終段階に入り、アバカンが内外からの危機に瀕する。）ヴィクトルがイリアス軍（あるいは自身の派閥・強化された眷属）と共に首都を襲撃。彼が設置した自然魔法無効化装置が広範囲に作動し、ロディニアの魔法使いや騎士団は大幅に弱体化する。  
9. **マグヌスの最期** : マグヌスがヴィクトルと対峙。自然魔法が使えない絶望的な状況下で、知略と残された僅かな力（あるいは未知の力？）で抵抗するが、ヴィクトル（あるいはその配下）に敗北。死の間際に、長年探求してきた未知の闇魔法（デスティール？）を発現させようとするが、果たせずに絶命する。ラタノフはマグヌスの最後の指示で、重要な研究データと共にアバカンを脱出する。  
10. **ヴィクトルの暗躍（第一部終了時点）** : （首都攻防戦とマグヌス殺害後）戦争が激化し、両国が疲弊した混乱に乗じて、ヴィクトルはイリアス軍の一部、ロディニア第一騎士団の一部（ヴィクトル派）、魔法使い団の一部、そして一部の常闇の眷属や一般市民を扇動し、一斉蜂起させる。彼らはイリアス南部の工業街「ギアヘイム」を制圧し、そこを拠点とする。**（ここで第一部が終了し、3年後の第二部へ続く）**
